# ES6 Arrow Functions


---

## Content

ES6 **arrow** functions (`=>`) represent function **expressions** in shorter syntax. Arrow functions turn 
```javascript
function(arguments){expression}
```
into:
```javascript
arguments => expression
```

For example:
```javascript
// with ES5
let evens = odds.map(
    function(n){return n + 1}
  );

// with ES6
let evens = odds.map(n => n + 1);
```
