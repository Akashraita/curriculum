# Output to the console

---
## Content

`console.*` methods display data to the console. For instance, `console.log` prints text or variables:

```js
console.log("Message");
// Message
```

```js
console.log("Username: " + username);
// Username: enkilondon
```

Other methods include:

- `console.warn("Warning")`
- `console.error("Error!")`
- `console.trace()`
