# Basic Data Types (int, double)

---

## Content

Integers are whole numbers such as 1, 2, 30, 9999:
```cpp
int number = 10;
std::cout << number;
// 10
```

### double
---

Floating-point numbers (double) are numbers that contain a decimal point.
```cpp
double number = 10.5;
std::cout << number; 
// 10.5
```
