# Basic Data Types (char, string)

---

## Content

`char` is used to represent a single character, such as 'a', 'Z', '2':

```cpp
char letter = 'a';
printf(letter);
// a
```

### string

`strings` represent a sequence of characters, such as "hello" or "Enki":

```cpp
std::string greeting = "Hello, world!";
printf(greeting);
// Hello, world!
```
