# Basic Data Types in R

---

## Content

Next you need to define similar functions for the other calculations that you want to support (e.g. subtraction, multiplication, division).
```r
subtract <- function(x, y) {
  return(x - y)
}
multiply <- function(x, y) {
  return(x * y)
}
divide <- function(x, y) {
  return(x / y)
}
```
