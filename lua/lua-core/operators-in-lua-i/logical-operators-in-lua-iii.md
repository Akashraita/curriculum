# The "Or" Operator

---
## Content

The `or` operator returns `true` if **either of its operands are true**, and `false` otherwise.

Here are some examples:
```lua
-- or operator
print(true or true)    --> true
print(true or false)   --> true
print(false or false)  --> false
```
